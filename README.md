# SQL Injection Practice Lab

교육용 SQL injection 실습 사이트다. 실제 서버나 데이터베이스 없이 브라우저 내부 mock data만 사용해 SQLi 원리와 방어법을 비교 학습하도록 구성했다.

## 파일 구성

- `HARNESS_PROMPT.md`: 강화된 하네스 엔지니어링 프롬프트
- `index.html`: 사이트 구조와 학습 콘텐츠
- `styles.css`: 레이아웃, 반응형, 모션
- `script.js`: 시뮬레이션 로직과 입력 가드레일
